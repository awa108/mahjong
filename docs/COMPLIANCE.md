### 功能介绍

本小程序是一款面向亲友的**休闲四人麻将**对战工具，核心特点：

- **纯娱乐，不涉真钱**：无充值入口、无提现功能、无虚拟代币、无积分兑换
- **房间制**：创建房间 → 邀请好友 → 进入游戏，像线下约局一样
- **权威防作弊**：洗牌、发牌、胜负判定全在服务端，客户端无法作弊
- **规则透明**：采用简化四人麻将规则（136 张牌，支持吃碰杠胡）

### 目标用户

- 熟悉麻将规则、希望与远方亲友线上娱乐的用户
- 仅通过微信好友关系分享房间码组局，不提供陌生人匹配

### 核心原则

```
本小程序严格遵循《微信小程序平台运营规范》第 3.4 条"棋牌类目规范"：
不做赌博、不接充值、不提现、不积分兑钱、不房卡代理、不广告激励。
```

## 4. 审核测试说明

> 提交审核时，将以下信息填入微信公众平台的「测试信息」中。

### 4.1 测试账号

| 项目 | 内容 |
|------|------|
| 测试 OpenID | （审核团队无需登录，可输入任意昵称直接进入） |
| 备注 | 本小程序开发环境走 mock login，输入昵称即可自动登录，无需真实微信账号 |

### 4.2 测试入口

1. 打开小程序，进入首页
2. 首页中央有「🀄 麻将」标题和「休闲四人麻将 · 好友约局」副标题
3. 在输入框中输入昵称（如"测试玩家"），点击「进入大厅」按钮
4. 进入大厅页面

### 4.3 核心功能测试路径

#### 路径一：创建房间 → 准备 → 开始游戏

1. 在大厅页面，点击「创建房间」
2. 系统自动生成 6 位房间码，跳转到房间页
3. 房间页显示 4 个座位（东南西北），当前玩家在"东"位
4. 点击「准备」按钮
5. 需要 4 人全准备才能开始（由于审核环境只有 1 人，此时可以看到房间码和准备状态，但无法开始）
6. **快速验证**：开发环境可模拟多人测试，详见测试文档

#### 路径二：测试房间码复制分享

1. 在房间页，点击右上角「分享」按钮（或 ··· 菜单）
2. 可以分享给微信好友
3. 好友点击分享卡片后，在首页输入昵称，在大厅输入 6 位房间码即可加入

#### 路径三：对局体验（需要 4 人）

1. 房主点击「开始游戏」→ 跳转到对局页
2. 对局页展示：本家手牌（底部）、弃牌河、对手面板、操作按钮
3. 轮到本家时，点击手牌选中 → 点击「出牌」
4. 其他玩家出牌时，本家可响应：吃/碰/杠/胡/过
5. 胡牌或流局后，跳转结算页展示结果

### 4.4 无支付功能声明

本小程序**没有任何支付功能**。代码中全局搜索关键词 `wx.requestPayment`、`支付`、`充值`、`购买`、`pay`、`charge`、`coin` 等均为零命中。

### 4.5 页面路径清单

| 页面 | 路径 | 用途 |
|------|------|------|
| 首页 | `pages/index/index` | 昵称输入 + 进入大厅 |
| 大厅 | `pages/lobby/lobby` | 创建/加入房间 |
| 房间 | `pages/room/room` | 座位展示 + 准备 + 开始 |
| 对局 | `pages/game/game` | 手牌 + 出牌 + 吃碰杠胡 |
| 结算 | `pages/result/result` | 胜者 + 分差 + 事件日志 |
