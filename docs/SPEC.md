# SPEC — 麻将小程序 MVP 技术规格

## 目标
微信原生小程序 4 人实时联网麻将，纯休闲，不涉及真钱。权威服务器做所有游戏逻辑校验。

## 技术栈
- 前端：微信原生小程序，TypeScript
- 后端：Node.js + TypeScript，CloudBase 云托管（WebSocket 常驻）+ 云函数（HTTP 短请求）
- 数据库：CloudBase 云开发文档型 DB
- 规则引擎：纯函数 + 状态机，前后端共享

## 规则（simple4）
- 136 张，无花牌
- 吃/碰/明杠/暗杠/补杠/点炮胡/自摸胡
- 第一版番型：基础胡（4 面子 + 1 雀头）或七对，简单计分

## 权威服务器原则
- 洗牌、发牌、校验、胜负：全在服务端
- 客户端只展示 + 提交操作
- 手牌按座位裁剪，客户端不可窃取他家
- 前端不存储任何密钥
